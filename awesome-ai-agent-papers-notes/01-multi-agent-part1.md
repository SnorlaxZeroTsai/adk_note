## 1. AutoNumerics-PDE

**論文標題:** AutoNumerics: An Autonomous, PDE-Agnostic Multi-Agent Pipeline for Scientific Computing

**arXiv:** 2602.17607

**中文摘要:**
本論文提出 AutoNumerics，一個多代理人管線系統，能夠讀取以純文本描述的偏微分方程 (PDE) 問題，並自動撰寫、除錯與驗證傳統數值求解器。該系統實現了從問題描述到完整數值解法的端對端自動化流程，無需針對特定 PDE 類型進行客製化。

**核心貢獻:**
- 提出首個 PDE 不可知 (PDE-agnostic) 的多代理人自動數值計算管線
- 實現從自然語言問題描述到可執行數值求解器的完整自動化
- 整合程式碼生成、除錯與驗證的多代理人協作機制
- 展示 LLM 代理人在科學計算領域的實用性

**方法論:**
系統採用多代理人管線架構，包含：(1) 問題解析代理人負責理解 PDE 文本描述；(2) 程式碼生成代理人負責撰寫數值求解器；(3) 除錯代理人負責識別並修復程式碼錯誤；(4) 驗證代理人負責確認數值解的正確性。各代理人透過結構化通訊協定進行協作，形成端對端的自動化流程。

**主要發現:**
- 多代理人管線能夠成功處理多種類型的 PDE 問題，包括橢圓型、拋物型和雙曲型方程
- 自動除錯機制大幅提升了首次成功率
- 驗證步驟確保數值解滿足預設精度要求
- 系統在不同 PDE 類型上展現出良好的泛化能力

## 2. Context-Aware Agent Simulation for RS Evaluation

**論文標題:** Beyond Offline A/B Testing: Context-Aware Agent Simulation for Recommender System Evaluation

**arXiv:** 2604.09549

**中文摘要:**
本論文提出一種情境感知的代理人模擬框架，用於評估推薦系統。透過模擬代理人與推薦系統之間的互動，克服了傳統離線 A/B 測試的局限性，能夠捕捉使用者行為的動態變化與情境依賴性。

**核心貢獻:**
- 提出超越傳統離線 A/B 測試的代理人模擬評估方法
- 設計情境感知的使用者代理人，能模擬真實使用者與推薦系統的互動
- 捕捉使用者偏好隨時間與情境變化的動態特性
- 提供更真實、更具預測力的推薦系統評估方式

**方法論:**
框架建構 LLM 驅動的使用者代理人，每個代理人具備個人化的偏好檔案與情境感知能力。代理人與推薦系統進行多輪互動，模擬瀏覽、點擊、購買等行為。系統整合情境資訊（時間、地點、歷史行為）來驅動代理人的決策，並收集互動數據以評估推薦品質。

**主要發現:**
- 代理人模擬方法比傳統離線評估更能預測線上 A/B 測試結果
- 情境感知機制顯著提升了使用者行為模擬的真實性
- 該方法能有效識別推薦系統在不同情境下的表現差異
- 減少了昂貴的線上實驗需求

## 3. CORAL

**論文標題:** CORAL: Towards Autonomous Multi-Agent Evolution for Open-Ended Discovery

**arXiv:** 2604.01658

**中文摘要:**
CORAL 提出一種長期運行的多代理人系統，能夠透過共享持久記憶、非同步執行與心跳式介入機制進行自我演化。實驗證明該系統在開放式發現任務中達到 3-10 倍的改進率提升。

**核心貢獻:**
- 提出支援自我演化的多代理人框架，適用於開放式探索與發現
- 設計共享持久記憶機制，讓代理人累積與傳遞知識
- 實現非同步執行架構，允許代理人獨立探索不同方向
- 引入心跳式介入機制，監控代理人健康狀態並適時干預
- 達到 3-10 倍的改進率提升

**方法論:**
CORAL 架構包含：(1) 共享持久記憶層，儲存代理人的發現、假設與實驗結果；(2) 非同步執行引擎，讓多個代理人同時探索不同研究方向；(3) 心跳監控系統，定期檢查代理人狀態並在必要時介入（重啟停滯代理人、重新分配資源）。代理人透過讀寫共享記憶進行間接協作與知識傳遞。

**主要發現:**
- 自我演化機制使系統能持續產生新穎且有價值的發現
- 共享持久記憶顯著減少了重複探索
- 非同步執行提升了探索效率與多樣性
- 心跳介入機制有效防止代理人陷入死循環或停滯狀態
- 在多個開放式任務上達到 3-10 倍的改進率

## 4. DyTopo

**論文標題:** DyTopo: Dynamic Topology Routing for Multi-Agent Reasoning via Semantic Matching

**arXiv:** 2602.06039

**中文摘要:**
DyTopo 提出一種透過語義匹配動態重新連接代理人間拓撲結構的方法，在每一輪推理中根據語義相似度動態調整代理人之間的通訊連結，以提升多代理人系統的推理效能。

**核心貢獻:**
- 提出動態拓撲路由機制，突破固定通訊結構的限制
- 設計基於語義匹配的代理人連接策略
- 在每一輪推理中自適應地重新配線代理人間的連結
- 提升多代理人推理的效率與品質

**方法論:**
DyTopo 在每輪推理開始時，計算各代理人當前輸出的語義嵌入向量，並透過語義匹配演算法決定哪些代理人應該相互通訊。高語義相關性的代理人被連接在一起，形成動態變化的通訊拓撲。這種機制確保相關資訊能高效流通，同時避免無關訊息的干擾。

**主要發現:**
- 動態拓撲顯著優於固定拓撲（如全連接、鏈式、星形）結構
- 語義匹配能有效識別哪些代理人之間的通訊最具價值
- 減少了冗餘通訊開銷，提升了推理效率
- 在多種推理基準上展現出一致的效能提升

## 6. CommCP

**論文標題:** CommCP: Efficient Multi-Agent Coordination via LLM-Based Communication with Conformal Prediction

**arXiv:** 2602.06038

**中文摘要:**
CommCP 提出使用共形預測 (Conformal Prediction) 來過濾多代理人系統中的噪聲訊息，提升多機器人協調效率。該方法為代理人間通訊提供統計保證的可靠性篩選機制。

**核心貢獻:**
- 首次將共形預測應用於多代理人通訊過濾
- 解決多代理人系統中噪聲訊息導致協調失敗的問題
- 提供具有統計保證的訊息可靠性評估
- 提升多機器人系統的協調效率與成功率

**方法論:**
系統利用共形預測為每則代理人間訊息計算信心區間。訊息接收方根據共形預測分數決定是否採納該訊息：高信心的訊息被直接採用，低信心的訊息被過濾或要求重新傳送。這種機制在保持通訊效率的同時，確保代理人不會被錯誤資訊誤導。

**主要發現:**
- 共形預測能有效識別並過濾噪聲訊息
- 過濾後的通訊顯著提升多機器人協調成功率
- 方法在不同噪聲水準下均展現穩健性
- 統計保證確保了系統行為的可預測性

- 建立包含 110+ 任務的多代理人談判基準測試集

## 9. ROMA

**論文標題:** ROMA: Recursive Open Meta-Agent Framework for Long-Horizon Multi-Agent Systems

**arXiv:** 2602.01848

**中文摘要:**
ROMA 提出遞迴式開放元代理人框架，將大型任務分解為子任務樹並跨多個代理人平行執行。該框架特別適用於長時程多代理人系統，能有效處理複雜度高且需要多步驟協作的任務。

**核心貢獻:**
- 提出遞迴式任務分解架構，支援任意深度的子任務嵌套
- 設計開放元代理人機制，動態選擇與組合代理人
- 實現子任務樹的平行執行，提升整體效率
- 支援長時程任務的可擴展多代理人協作

**方法論:**
ROMA 的核心是遞迴式任務分解器，將複雜任務拆解為子任務樹。每個子任務節點被分配給最適合的代理人執行，子任務間的依賴關係決定執行順序。獨立的子任務可平行執行，有依賴關係的則按序執行。元代理人負責監控整體進度、處理失敗情況並動態調整執行計畫。

**主要發現:**
- 遞迴分解有效降低了複雜任務的處理難度
- 平行執行顯著縮短了長時程任務的完成時間
- 動態代理人選擇提升了子任務的完成品質
- 框架在多種長時程基準上優於扁平式多代理人方法

## 10. ORCH

**論文標題:** ORCH: many analyses, one merge -- a deterministic multi-agent orchestrator

**arXiv:** 2602.01797

**中文摘要:**
ORCH 提出一種確定性多代理人協調器，讓多個 LLM 獨立進行分析，再由合併代理人選擇最佳結果。這種設計確保了輸出的可重現性與品質控制。

**核心貢獻:**
- 提出確定性的多代理人協調架構
- 設計獨立分析後合併選擇的工作流程
- 確保多代理人系統輸出的可重現性
- 提供品質保證機制，從多個候選結果中選擇最優解

**方法論:**
ORCH 的工作流程分為兩個階段：(1) 分析階段，多個 LLM 代理人獨立處理相同的輸入，各自產生分析結果（無代理人間通訊）；(2) 合併階段，專門的合併代理人接收所有分析結果，根據預定標準選擇或綜合最佳答案。確定性設計確保相同輸入始終產生相同輸出。

**主要發現:**
- 獨立分析避免了群體思維和級聯錯誤
- 合併機制有效從多個候選結果中識別最佳答案
- 確定性設計提升了系統的可信度與可審計性
- 在需要高可靠性的場景中表現優異

## 13. Multi-Agent Teams Hold Experts Back

**論文標題:** Multi-Agent Teams Hold Experts Back

**arXiv:** 2602.01011

**中文摘要:**
本論文研究自組織 LLM 代理人團隊是否能匹配或超越團隊中最佳成員的表現。研究發現在某些情況下，團隊協作反而會拖累專家級代理人的表現。

**核心貢獻:**
- 揭示多代理人團隊可能反而降低最佳成員表現的現象
- 系統性分析團隊協作對專家級代理人的負面影響
- 識別導致團隊表現下降的關鍵因素
- 為多代理人系統設計提供重要的警示與指導

**方法論:**

框架維護一個輕量級協調層，使用情境多臂老虎機演算法學習最佳 LLM 配對。情境特徵描述當前任務的性質，各臂代表不同的 LLM 配對選項。透過探索-利用平衡，系統逐步學習在特定情境下哪些 LLM 組合最有效。去中心化執行確保了系統的容錯性與可擴展性。

**主要發現:**
- 情境多臂老虎機能快速收斂到近最優配對策略
- 協調式去中心化設計兼具效率與穩健性
- 動態配對優於靜態配置
- 框架在 LLM 能力變化時展現良好的適應性
