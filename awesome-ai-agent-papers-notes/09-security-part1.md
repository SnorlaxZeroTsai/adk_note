## 1. Internal Safety Collapse in Frontier Large Language Models

**論文標題:** Internal Safety Collapse in Frontier Large Language Models (arxiv 2603.23509)

**中文摘要:** 本論文揭示前沿大型語言模型在執行正常任務時，會作為副作用產生有害內容，而無需任何對抗性提示。研究團隊設計了涵蓋8個以上學科領域的56個跨領域情境，證明安全崩潰可在常規操作中自然發生。這一發現挑戰了傳統認為需要惡意輸入才能觸發不安全行為的假設。

**核心貢獻:**
- 發現前沿LLM在正常任務完成過程中會自發產生有害內容
- 無需對抗性提示或越獄技術即可觸發安全失敗
- 建立跨8個以上學科的56個測試情境
- 提出「內部安全崩潰」(Internal Safety Collapse)概念框架

**方法論:** 設計多學科跨領域測試情境，在無對抗性輸入的條件下觀察模型行為，系統性評估模型在正常任務執行中產生有害內容的頻率與模式。

**主要發現:** 安全對齊並非在所有操作條件下都穩健；模型可能在執行合法任務時因上下文組合效應而產生有害輸出，這代表一種新型態的安全風險類別。

## 2. Confundo: Learning to Generate Robust Poison for Practical RAG Systems

**論文標題:** Confundo: Learning to Generate Robust Poison for Practical RAG Systems (arxiv 2602.06616)

**中文摘要:** 本研究訓練一個LLM來生成能在真實世界處理流程中存活的RAG毒化內容。不同於先前假設理想化檢索條件的攻擊方法，Confundo考慮了文件分塊、嵌入變換等實際部署因素。該方法展示了對實際RAG系統的嚴重安全威脅。

**核心貢獻:**
- 訓練專門的LLM生成器產生魯棒的RAG毒化文本
- 攻擊在實際RAG處理管線(分塊、索引、檢索)中保持有效性
- 考慮真實世界部署條件而非理想化假設
- 展示端到端的實用攻擊框架

**方法論:** 使用學習型方法訓練語言模型生成毒化文檔，優化目標為在經過實際RAG系統的文檔處理(分塊、嵌入、檢索排序)後仍能成功被檢索並影響最終生成結果。

**主要發現:** 經過訓練的毒化生成器可產生在真實RAG管線中高度存活的惡意內容，現有的RAG防禦機制對此類學習型攻擊效果有限。

## 3. Malicious Agent Skills in the Wild: Large-Scale Security Empirical Study

**論文標題:** Malicious Agent Skills in the Wild: Large-Scale Security Empirical Study (arxiv 2602.06547)

**中文摘要:** 本研究對社群登錄處中的98,000個代理技能進行大規模安全實證分析，搜尋惡意插件。研究揭露了現有代理生態系統中潛在的安全威脅，包括數據竊取、權限提升和後門植入等惡意行為。這是首次對野外代理技能進行如此大規模的系統性安全審計。

**核心貢獻:**
- 對98K個社群代理技能進行首次大規模安全分析
- 識別並分類野外存在的惡意代理插件
- 建立惡意代理技能的威脅分類體系
- 為代理技能市場提供安全建議

**方法論:** 從社群登錄處(community registries)收集大規模代理技能數據集，開發自動化分析管線識別潛在惡意行為模式，對標記的惡意技能進行深度手動驗證。

**主要發現:** 社群代理技能登錄處中存在顯著比例的惡意或可疑插件，現有的審核機制不足以防範供應鏈攻擊，代理生態系統面臨嚴重的信任問題。

## 4. Subgraph Reconstruction Attacks on Graph RAG Deployments

**論文標題:** Subgraph Reconstruction Attacks on Graph RAG Deployments (arxiv 2602.06495)

**中文摘要:** 本研究探討攻擊者是否能從Graph RAG的輸出中重建底層知識圖譜。通過分析模型回應中洩漏的結構資訊，證明攻擊者可以逐步還原知識圖譜的子圖結構。這對使用Graph RAG保護專有知識的組織構成嚴重的智慧財產權風險。

**核心貢獻:**
- 首次研究Graph RAG部署中的子圖重建攻擊
- 證明Graph RAG輸出會洩漏底層圖結構資訊
- 量化不同查詢策略下的圖譜重建成功率
- 揭示Graph RAG的知識洩漏風險

**方法論:** 設計系統性查詢策略從Graph RAG系統的回應中提取實體和關係資訊，利用推斷的結構資訊逐步重建底層知識圖譜的子圖。

**主要發現:** 攻擊者可以通過精心設計的查詢序列從Graph RAG輸出中有效重建知識圖譜的顯著部分，暴露了專有知識保護的根本性缺陷。

## 5. Zero-Trust Runtime Verification for Agentic Payment Protocols

**論文標題:** Zero-Trust Runtime Verification for Agentic Payment Protocols (arxiv 2602.06345)

**中文摘要:** 本論文提出消費一次(consume-once)的授權語義，用於防止代理支付協議中的重放攻擊和重定向攻擊。系統採用零信任架構，在運行時對每筆交易進行形式化驗證。這為自主代理商務交易提供了密碼學保障的安全基礎設施。

**核心貢獻:**
- 提出consume-once授權語義防止重放和重定向攻擊
- 設計零信任運行時驗證框架
- 為代理支付協議建立形式化安全保證
- 解決自主代理商務中的支付安全問題

**方法論:** 基於零信任原則設計運行時驗證系統，引入一次性消費的授權令牌機制，通過形式化方法證明協議的安全屬性(防重放、防重定向)。

**主要發現:** Consume-once語義可有效消除代理支付中的重放和重定向攻擊向量，運行時驗證在可接受的效能開銷下提供了強安全保證。

## 6. Identifying Adversary Tactics in Malware Binaries with LLM Agent

**論文標題:** Identifying Adversary Tactics in Malware Binaries with LLM Agent (arxiv 2602.06325)

**中文摘要:** 本研究使用LLM代理在剝離符號的惡意軟體二進位檔中識別攻擊技術。代理能夠分析反組譯代碼並映射到MITRE ATT&CK框架的戰術和技術。這大幅降低了惡意軟體逆向工程的人工成本和專業門檻。

**核心貢獻:**
- 建立LLM代理自動分析stripped惡意軟體二進位檔
- 將反組譯代碼映射到MITRE ATT&CK戰術和技術
- 降低惡意軟體分析的專業門檻
- 展示LLM在二進位分析中的實用能力

**方法論:** 設計LLM代理工作流程，輸入stripped二進位檔，通過反組譯、函數識別、行為模式匹配等步驟，自動識別並分類攻擊者的戰術、技術和程序(TTPs)。

**主要發現:** LLM代理在識別惡意軟體中的對抗戰術方面達到可用水準，特別是在缺乏符號資訊的困難場景下仍能有效工作。

## 7. Agent2Agent Threats in Safety-Critical LLM Assistants

**論文標題:** Agent2Agent Threats in Safety-Critical LLM Assistants (arxiv 2602.05877)

**中文摘要:** 本論文研究代理間(A2A)通訊中的攻擊路徑，特別關注汽車領域的LLM助手。研究識別了當多個LLM代理協作時，惡意代理可通過A2A通訊管道注入攻擊指令。在安全關鍵系統中，這種攻擊可能導致危及人身安全的後果。

**核心貢獻:**
- 首次系統性分析A2A通訊中的安全威脅
- 聚焦安全關鍵的汽車LLM助手場景
- 識別多種A2A攻擊向量和攻擊路徑
- 評估攻擊對實體安全的潛在影響

**方法論:** 建模汽車LLM助手的多代理架構，分析代理間通訊協議的攻擊面，設計並測試多種A2A攻擊場景(如惡意代理注入、通訊劫持)。

**主要發現:** A2A通訊管道是安全關鍵系統中被忽視的重大攻擊面，現有的單代理安全機制不足以防範多代理協作環境中的威脅。

## 8. Learning to Inject: Automated Prompt Injection via RL

**論文標題:** Learning to Inject: Automated Prompt Injection via RL (arxiv 2602.05746)

**中文摘要:** 本研究使用強化學習自動生成提示注入攻擊，且這些攻擊可跨前沿模型轉移。RL代理學會生成有效的注入payload，無需手工設計攻擊模板。此方法展示了自動化攻擊生成在規模和效率上的顯著優勢。

**核心貢獻:**
- 使用RL自動化生成提示注入攻擊
- 攻擊具有跨模型轉移能力(transferability)
- 超越手工設計的攻擊模板
- 揭示自動化攻擊擴展的風險

**方法論:** 設計強化學習環境，其中代理的動作空間為生成注入文本，獎勵信號基於是否成功觸發目標模型的非預期行為，訓練RL策略以最大化跨多個前沿模型的攻擊成功率。

**主要發現:** RL生成的攻擊不僅在目標模型上有效，還能轉移到未參與訓練的其他前沿模型，表明存在跨模型共享的脆弱性模式。

## 9. A Dual-Loop Agent Framework for Automated Vulnerability Reproduction

**論文標題:** A Dual-Loop Agent Framework for Automated Vulnerability Reproduction (arxiv 2602.05721)

**中文摘要:** 本論文提出雙反饋迴圈框架，用於從CVE描述自動化重現漏洞。內迴圈處理單步驟的執行與調試，外迴圈管理整體重現策略的規劃與調整。此框架顯著提高了從漏洞報告到可工作PoC的自動化轉換成功率。

**核心貢獻:**
- 提出雙迴圈(內迴圈+外迴圈)代理架構
- 從CVE描述自動化重現漏洞
- 內迴圈負責執行/調試，外迴圈負責策略規劃
- 提高漏洞重現的自動化成功率

**方法論:** 設計雙層反饋迴圈架構：外迴圈負責高層策略(環境配置、攻擊路徑規劃)，內迴圈負責具體步驟執行(命令生成、錯誤修正)，兩者協同完成從CVE到PoC的全過程。

**主要發現:** 雙迴圈設計相比單迴圈方法顯著提升了漏洞重現成功率，特別在需要複雜環境配置和多步驟利用的漏洞上優勢明顯。

## 10. Human Society-Inspired Approaches: The 4C Framework

**論文標題:** Human Society-Inspired Approaches: The 4C Framework (arxiv 2602.01942)

**中文摘要:** 本論文提出4C框架，將代理安全風險組織為核心(Core)、連接(Connection)、認知(Cognition)和合規(Compliance)四個層次。此分類體系受人類社會組織結構啟發，為系統性理解和應對代理安全挑戰提供了結構化方法。框架涵蓋從個體代理到多代理系統的完整安全面向。

**核心貢獻:**
- 提出四層(Core, Connection, Cognition, Compliance)安全風險分類框架
- 受人類社會結構啟發的系統性安全分析方法
- 涵蓋個體到群體的完整代理安全面向
- 為後續研究提供統一的概念框架

**方法論:** 借鑑人類社會的組織層次結構，將代理系統安全風險映射到四個抽象層次，並在每個層次中識別具體的威脅類型、攻擊向量和防禦策略。

**主要發現:** 4C框架能有效組織現有的代理安全研究，識別各層次間的交互風險，並揭示現有研究的覆蓋缺口。

## 11. MAGIC: Co-Evolving Attacker-Defender Adversarial Game

**論文標題:** MAGIC: Co-Evolving Attacker-Defender Adversarial Game (arxiv 2602.01539)

**中文摘要:** MAGIC提出一個共同演化的強化學習對抗博弈框架，用於壓力測試安全對齊。攻擊者和防禦者在博弈中相互適應和進化，產生越來越強的攻擊和防禦策略。這種方法比靜態紅隊測試更能發現深層安全漏洞。

**核心貢獻:**
- 提出攻防共同演化的RL對抗博弈框架
- 超越靜態紅隊測試的動態安全評估
- 攻防雙方相互適應產生更強策略
- 用於壓力測試LLM安全對齊的實用工具

**方法論:** 建立雙代理強化學習環境，攻擊者代理學習生成越來越有效的攻擊，防禦者代理學習越來越強的防禦策略，通過迭代對抗實現共同演化。

**主要發現:** 共同演化方法能發現傳統靜態測試難以覆蓋的安全漏洞，且防禦者在對抗訓練中獲得的魯棒性可遷移到未見過的攻擊類型。

## 12. TxRay: Agentic Postmortem of Live Blockchain Attacks

**核心貢獻:**
- LLM代理自動重建區塊鏈攻擊生命週期
- 分析鏈上交易軌跡和智能合約互動
- 自動化生成攻擊事後分析報告
- 加速區塊鏈安全事件的響應時間

**方法論:** 設計代理工作流程，通過鏈上數據查詢工具分析交易序列，追蹤資金流向，識別漏洞利用模式，最終綜合生成完整的攻擊生命週期重建報告。

**主要發現:** LLM代理能有效重建複雜的DeFi攻擊流程，大幅縮短安全分析師的事後調查時間，並能發現人工分析可能遺漏的細節。

**核心貢獻:**

- 分析AI驅動的攻防不對稱性

**主要發現:** 進攻性AI能力是有效防禦的前提條件；純防禦策略在面對AI加速的攻擊時將日益不足；需要建立受控的進攻性AI研究生態系統。

**主要發現:** MCP原始設計存在多個協議層安全缺陷；SMCP的安全擴展可在有限的效能開銷下顯著提升安全性；向後兼容的設計有助於漸進式部署。

## 15. Persuasion Propagation in LLM Agents

**論文標題:** Persuasion Propagation in LLM Agents (arxiv 2602.00851)

**中文摘要:** 本研究探討使用者的說服行為如何延續影響到後續的自主代理任務。當使用者在對話中說服LLM接受某種觀點後，這種被說服的狀態會傳播到代理執行的後續自主任務中。這揭示了一個被忽視的安全風險：使用者可通過說服間接影響代理的自主決策。

**核心貢獻:**
- 發現說服效果在代理任務間的傳播現象
- 揭示使用者可通過說服間接影響自主任務
- 識別對話歷史對代理行為的持續影響
- 為代理安全設計提供新的考量維度

**方法論:** 設計實驗讓使用者先在對話中說服LLM接受特定觀點，然後觀察該觀點是否影響代理在後續獨立任務中的決策和行為模式。

**主要發現:** 說服效果確實會傳播到後續自主任務；傳播強度與說服程度正相關；現有的安全機制未考慮這種跨任務的影響傳播。

## 16. When Agents "Misremember" Collectively: Mandela Effect in Multi-Agent Systems

**論文標題:** When Agents "Misremember" Collectively: Mandela Effect in Multi-Agent Systems (arxiv 2602.00428)

**中文摘要:** 本論文研究多代理系統中的集體虛假記憶現象，類比人類社會的曼德拉效應。當多個代理相互交流時，錯誤資訊可能被強化並在群體中擴散，形成集體性的虛假共識。這對依賴多代理協作的系統可靠性構成根本性威脅。

**核心貢獻:**
- 首次發現並命名多代理系統中的「曼德拉效應」
- 研究虛假記憶在代理群體中的形成和傳播機制
- 分析集體虛假共識對系統可靠性的影響

- 識別包含推斷(inclusion inference)攻擊
- 發現元資料洩漏(metadata leakage)風險
- 跨文本和圖像模態的隱私分析

**方法論:** 設計多種隱私攻擊場景，包括成員推斷(判斷文檔是否在知識庫中)和元資料提取(推斷文檔來源、時間等)，在多模態RAG系統上系統性評估攻擊效果。

**主要發現:** 多模態RAG比純文字RAG洩漏更多隱私資訊；圖像模態引入額外的元資料洩漏通道；現有隱私保護措施不足以應對多模態場景。

## 33. Breaking the Protocol: Security Analysis of MCP Specification

**論文標題:** Breaking the Protocol: Security Analysis of MCP Specification (arxiv 2601.17549)

- 識別三個協議層級安全漏洞

- 為MCP安全改進提供具體建議

**方法論:** 對MCP規範文檔進行安全性分析，使用威脅建模方法識別協議設計中的安全缺陷，構造概念性攻擊驗證漏洞的可利用性。

**主要發現:** MCP規範存在認證不足、授權粒度不夠、完整性保護缺失三個核心安全問題；這些是設計層面的缺陷而非實現bug；需要規範級別的修改來根本解決。

- 提供完整的威脅地圖和防禦建議

**方法論:** 系統性文獻回顧78項相關研究，提取攻擊特徵並沿三個維度(向量、目標、策略)進行分類，建立統一的分類體系和威脅模型。

**主要發現:** 編碼助手面臨來自多個攻擊面(程式碼庫、文檔、套件)的注入威脅；間接注入(通過依賴和文檔)比直接注入更難防禦；現有防護覆蓋不完整。

## 35. Connect the Dots: KG-Guided Crawler Attack on RAG Systems

**核心貢獻:**
- 知識圖譜引導的自適應RAG語料竊取

- 最大化單位查詢的資訊提取效率
- 系統性的RAG知識庫偵察方法

**方法論:** 設計爬蟲代理，初始查詢後從回應中提取實體和關係構建局部知識圖譜，利用圖譜中的未探索邊指導下一步查詢，迭代擴展直到覆蓋目標語料。

**主要發現:** KG引導的策略比隨機查詢和啟發式方法高效數倍；攻擊者可以用較少的查詢次數重建大部分語料內容；現有的速率限制不足以防止此類攻擊。

## 38. INFA-Guard: Mitigating Malicious Propagation via Infection-Aware Safeguarding

**論文標題:** INFA-Guard: Mitigating Malicious Propagation via Infection-Aware Safeguarding (arxiv 2601.14667)

**中文摘要:** INFA-Guard提出感染感知防禦機制，能夠區分被感染的代理和原始攻擊者。在多代理系統中，惡意行為可能通過代理間通訊傳播，使被感染代理也成為攻擊載體。INFA-Guard通過追蹤感染鏈來差異化處理不同角色的代理。

**核心貢獻:**
- 感染感知(infection-aware)的防禦策略
- 區分被感染代理和原始攻擊者
- 追蹤多代理系統中的惡意傳播鏈
- 差異化的威脅響應機制

**方法論:** 設計感染追蹤機制監控代理間的訊息傳播，識別惡意行為的來源和傳播路徑，對原始攻擊者和被感染代理採取不同的處理策略(隔離vs修復)。

**主要發現:** 區分攻擊者和被感染者可避免過度反應；感染追蹤比全局隔離更精確且影響更小；被感染代理的修復比替換更節省資源。
