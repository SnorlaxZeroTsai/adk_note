## 每篇筆記格式

每篇論文包含以下欄位：
- **論文標題** — 英文原始標題 + arXiv ID
- **中文摘要** — 2-3 句繁體中文摘要
- **核心貢獻** — 主要貢獻列點
- **方法論** — 技術方法概述
- **主要發現** — 研究結果與發現

## 主要研究趨勢

1. **多代理人協作架構** — 從固定拓撲走向動態路由、信心感知調度
2. **記憶系統** — 從靜態 RAG 走向自適應、分層、可學習的記憶架構
3. **評估方法** — 從結果導向走向過程感知、軌跡層級的評估
4. **工具使用** — 從靜態工具庫走向測試時工具演化、meta-tool
5. **安全防護** — 從被動防禦走向主動偵測、零信任架構
